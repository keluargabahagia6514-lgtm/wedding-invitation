# Wedding Invitation - 3D Website

Undangan pernikahan interaktif dengan animasi 3D menggunakan Three.js.

## ✨ Fitur

- 🚪 **Cover 3D** — Gerbang pernikahan terbuka dengan animasi
- 💍 **Cincin 3D** — Model cincin berputar (Three.js)
- 🌸 **Partikel Bunga** — Efek kelopak bunga berjatuhan
- ⭐ **Background Bintang** — Animasi bintang & partikel 3D
- 📸 **Galeri Carousel** — Foto pre-wedding dengan efek swipe
- ⏰ **Countdown** — Hitung mundur ke hari pernikahan
- 📝 **Buku Tamu** — Form ucapan & konfirmasi kehadiran
- 🎵 **Pemutar Musik** — Background music otomatis
- 🔮 **Glassmorphism** — Desain kartu transparan elegan
- 📱 **Responsive** — Tampilan optimal di HP & desktop

## 🛠️ Tech Stack

- HTML5 & CSS3 (Glassmorphism, Parallax)
- JavaScript (ES6+)
- Three.js (3D rendering)

## 🚀 Cara Pakai

```bash
# Clone repository
git clone https://github.com/keluargabahagia6514-lgtm/wedding-invitation.git
cd wedding-invitation

# Jalankan server lokal
python3 -m http.server 8888

# Buka di browser
# http://localhost:8888
```

## ✏️ Kustomisasi

| File | Yang Diubah |
|------|-------------|
| `index.html` | Nama, tanggal, lokasi, love story |
| `assets/images/` | Foto mempelai & galeri |
| `assets/music/` | Musik background (.mp3) |
| `css/style.css` | Warna & tema |

## 📂 Struktur Folder

```
├── index.html
├── css/style.css
├── js/
│   ├── intro3d.js      ← 3D pintu gerbang
│   ├── scene3d.js      ← Background 3D
│   └── app.js          ← Logic utama
└── assets/
    ├── images/
    ├── music/
    └── icons/
```
